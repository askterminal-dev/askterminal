Modifying Commands with Flags

Commands often accept flags (also called options) that modify their behavior. Flags start with a dash (-).

What Are Flags?

A flag is an extra instruction you add to a command. For example:

ls -l

The -l flag tells ls to show the "long" format with details.

Common Flag Patterns

Single letter flags use one dash:

ls -a

Show all files, including hidden ones (starting with a dot).

Combine multiple flags together:

ls -la

Same as ls -l -a — long format AND show hidden files.

Word flags use two dashes:

ls --all

Same as ls -a — the word version is easier to remember.

Flags with Values

Some flags take a value after them:

head -n 5 file.txt

The -n 5 tells head to show only 5 lines.

Getting Help

Most commands show their available flags with --help:

ls --help

Shows all flags and options for the ls command.

For detailed documentation, use the manual:

man ls

Opens the full manual. Press q to exit.

Practice

Try these variations of ls:

  1. ls — basic list
  2. ls -l — long format with details
  3. ls -a — include hidden files
  4. ls -la — both together
  5. ls -lh — long format with human-readable sizes
