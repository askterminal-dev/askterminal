Welcome to the Terminal

You have access to a real terminal, but don't worry — we've added some safeguards to help you learn safely.

Think of this panel and the command area below as your training wheels, helping you gain your balance as you learn. You can't accidentally break anything here.

What's Protected

  • You start with access to simple, safe commands only
  • Commands that could cause harm are blocked or require confirmation
  • Every command is explained before you run it
  • Color-coded safety indicators show you what's safe

Safety Indicators

Green — Safe commands that just display information
Yellow — Commands that make changes (blocked for beginners)
Red — Powerful commands that need extra care

Ready to Start?

Try typing pwd in the command area below — it shows your current location in the file system. Completely safe, just information.
