Having Conversations with Claude

Claude Code's interactive mode lets you have natural back-and-forth conversations while working on your code.

Starting Interactive Mode

Type claude with no arguments to start a conversation:

claude

Opens an interactive session in your current directory

You'll see Claude's prompt. Now you can type questions or requests naturally.

Asking Questions

In interactive mode, just type what you want to know:

  • "What does this project do?"
  • "Explain the main function in app.js"
  • "Why is this test failing?"
  • "How do I add a new route?"

Claude will read relevant files to give you informed answers.

Context Awareness

Claude remembers your conversation. You can refer back to earlier topics:

  • "Now fix the bug we discussed"
  • "Add tests for that function"
  • "Actually, use TypeScript instead"

Continuing Later

Resume your last conversation:

claude -c

Continue from where you left off

claude -r

Pick from a list of past conversations to resume

Exiting

When you're done, type one of these:

  • /exit or /quit — Leave interactive mode
  • Ctrl+C — Also exits (or stops current action)

Tips for Good Conversations

  • Be specific — "Fix the login bug" is better than "fix bugs"
  • Give context — Mention file names or function names when relevant
  • Iterate — If the first answer isn't right, clarify what you need
