Getting Started with Claude Code

Claude Code is an AI assistant that lives in your terminal. It can help you write code, understand projects, fix bugs, and run commands.

What is Claude Code?

Think of Claude Code as a knowledgeable programming partner that can:

  • Read and understand your code files
  • Write and edit code for you
  • Run terminal commands on your behalf
  • Search the web for documentation
  • Explain what's happening step by step

Installing Claude Code

If you don't have Claude Code installed yet, run this command:

npm install -g @anthropic-ai/claude-code

Installs Claude Code globally on your system

You'll need an Anthropic API key. Claude will prompt you to set it up on first run.

Starting Claude

There are two ways to use Claude Code:

claude

Start an interactive session — have a back-and-forth conversation

claude "your question here"

Ask a one-off question — get an answer and return to your prompt

Your First Interaction

Try starting Claude with a simple request:

claude "what files are in this directory?"

Claude will look at your files and describe what it finds

Watch what happens — Claude will show you when it uses tools like reading files or running commands. Everything is transparent.

Key Things to Know

  • Claude asks permission — Before making changes, Claude will ask
  • You can say no — Deny any action you're unsure about
  • It's reversible — Use git to undo changes if needed
  • Start small — Begin with questions before asking for edits
