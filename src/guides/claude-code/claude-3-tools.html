Understanding Claude's Tools

Claude Code uses "tools" to interact with your computer. When you see Claude using a tool, it's being transparent about what it's doing.

What Are Tools?

Tools are specific actions Claude can take. Instead of just talking, Claude can actually read files, write code, and run commands.

You'll see tool names appear as Claude works — this is good! It means you can see exactly what's happening.

Reading Tools

Read — Opens and reads a file's contents
Glob — Finds files matching a pattern (like *.js)
Grep — Searches file contents for specific text

These tools only look at things — they don't change anything.

Writing Tools

Write — Creates a new file or replaces an existing one
Edit — Makes targeted changes to part of a file

These tools modify your code. Claude will show you what it plans to change and ask for permission first.

Command Tools

Bash — Runs a terminal command

Claude can run commands like npm install or git status. It will show you the command before running it.

Research Tools

WebSearch — Searches the web for information
WebFetch — Retrieves content from a specific URL

Claude can look up documentation, examples, or current information online.

Why This Matters

Seeing the tools helps you understand what Claude is doing and why. If you see Claude using Read on several files before answering, it's gathering context. If you see Edit, it's about to change code.
