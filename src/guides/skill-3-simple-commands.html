Your First Commands

Let's learn some basic commands. These are all safe — they just display information or help you navigate. Nothing gets changed or deleted.

See Where You Are

pwd

Print Working Directory — shows your current location in the file system.

Look Around

ls

List files and folders in the current directory.

Move Around

cd foldername

Change Directory — move into a folder.

cd ..

Go up one level (to the parent folder).

cd ~

Go to your home folder.

Display Text

echo "Hello!"

Print a message to the screen.

Clear the Screen

clear

Clears the terminal and gives you a fresh screen.

Get Help

man ls

Show the manual for a command. Press q to exit.

Practice

Try these commands in order:

  1. pwd — see where you are
  2. ls — see what's here
  3. cd .. — go up one level
  4. ls — see what's in the parent folder
  5. cd ~ — go back home
