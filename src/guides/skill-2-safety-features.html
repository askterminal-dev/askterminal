Understanding Safety Features

Ask Terminal has built-in safeguards to help you learn without worry. Let's understand how they protect you.

Experience Levels

Your experience level controls which commands you can run. You can see and change your level by clicking the badge in the bottom-right corner.

  • Beginner — Only safe, read-only commands allowed
  • Intermediate — Most commands allowed, dangerous ones need confirmation
  • Advanced — Full access to all commands

Start as a Beginner. As you get comfortable, level up to unlock more commands.

Safety Indicators

Every command shows a colored dot indicating its safety level:

Green — Safe commands that only display information
Yellow — Commands that create or modify files
Red — Powerful commands that can delete files or change system settings

Blocked Commands

In Beginner mode, yellow and red commands are blocked. You'll see a message explaining why, and the Run button will be disabled.

This isn't a limitation — it's protection while you learn. You can always see what a command would do, even if you can't run it yet.

The Command Input Area

Notice that you type commands in the panel below the terminal, not directly in the terminal itself. This is intentional:

  • You see explanations before running anything
  • Safety checks happen automatically
  • You can review and edit before pressing Run

In Intermediate mode, you can enable typing directly in the terminal if you prefer.

Confirmation Dialogs

In Intermediate mode, dangerous commands (red) will ask for confirmation before running. You'll see exactly what the command does and can choose to proceed or cancel.
